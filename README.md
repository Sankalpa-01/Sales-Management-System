# 🛒 Sales Management System

A full-stack Sales Management System built with:
- ⚛️ **Frontend**: React.js + TypeScript + Tailwind CSS
- 🚀 **Backend**: Node.js (Express) + MySQL
- 🔐 Secure login & authentication
- 📦 Inventory and product management
- 📊 Sales tracking and dashboard visualization


## 📁 Frontend Pages

- Dashboard – Displays charts and tables for Users, Customers, Products, and Sales.
- Customers – Full CRUD UI for customers: search, add, delete, and view details.
- Products - Allows users to manage the inventory of products, including adding, updating, and deleting products.
- Sales - Allows users to manage sales transactions, including creating new sales and viewing existing sales.
- Payments - Allows users to process payments for sales and view the payment history
- Reports - It provides insights into sales performance through various charts and summary metrics

## 📁 Backend
# Techstacks
- Node.js
- Express.js
- MySQL
- Sequelize (or native MySQL2 driver)

## 🚀 Features

- 🔐 User authentication (Register/Login)
- 🧑‍💼 Manage users and roles
- 🧾 Track sales and sale items
- 📦 Manage products and inventory logs
- 💰 Record and view payments
- 📈 Dashboard with sales analytics and inventory alerts
- 📄 AI-generated summaries for performance/feedback (optional extension)
